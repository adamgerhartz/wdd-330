Notes Week 4

  • The first thing I've learned is that we should not add any event listeners inside of our HTML file. Instead, to separate concerns, you can do two things: (1) you could add an event listener in between the <script> element, or you could create an external file that has the event listener. Either way, the purpose is to separte HTML from JavaScript. This is considered a good practice.
