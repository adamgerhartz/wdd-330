  • I've learned that semicolons are not required in ES6, but it is good practice to do so.
  • The concat() method is used to concatenate two or more strings together
  • Using backticks ``, one is able to put a variable into a string like so: `My name is ${ name }`
  • The delete operator will only remove an item from an array, not remove the space allocated in that array. An "undefined" will be put in place of that item
  • Use the splice() method to delete and get rid of the space allocated in the array
  • Arrow functions are pretty cool
  • Initialize with a value is better that just declaring a variable in JavaScript. For example, do "let sum = 0;" instead of "let sum;"
